# Animatrona

**Умная библиотека аниме с GPU-транскодированием**

Desktop приложение для управления аниме-коллекцией с автоматическим подбором качества.

## Возможности

- **GPU-ускоренное транскодирование** — AV1/HEVC через NVENC, до 2x параллельных потоков
- **Автоподбор качества VMAF** — интеллектуальный выбор CQ для целевого качества 95+
- **Импорт с Shikimori** — автозаполнение метаданных (название, описание, постер, жанры)
- **Встроенный плеер** — поддержка ASS субтитров с кастомными шрифтами
- **Управление франшизами** — группировка связанных тайтлов
- **Кроссплатформенность** — Windows, macOS, Linux

## Системные требования

- **Windows:** 10/11 (x64)
- **macOS:** 11+ (Intel/Apple Silicon)
- **Linux:** Ubuntu 20.04+ или аналог
- **GPU:** NVIDIA GTX 1650+ для транскодирования (опционально)
- **RAM:** 8 GB минимум, 16 GB рекомендуется

## Быстрый старт

1. Скачайте и установите приложение
2. Откройте папку с аниме через **File → Import**
3. Нажмите **«Найти на Shikimori»** для импорта метаданных
4. Выберите эпизод и наслаждайтесь просмотром
